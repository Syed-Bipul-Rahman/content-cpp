# Build Your Own Telegram Bot

Bot API, long polling vs webhooks, inline keyboards.

## What you'll build

- Register a bot with BotFather
- Long-poll updates and respond
- Switch to webhooks behind a TLS endpoint
- Send text, photos, documents, inline keyboards
- Build a small conversational state machine

## The core idea

Telegram's Bot API is JSON over HTTPS. Once you have an HTTPS client (we'll build a small one), the bot is straightforward.
