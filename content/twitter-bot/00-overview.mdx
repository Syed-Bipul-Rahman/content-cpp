# Build Your Own Twitter/X Bot

OAuth 2.0, streaming, rate limits, content policy-aware posting.

## What you'll build

- Implement OAuth 2.0 with PKCE
- Post tweets via the v2 API
- Stream filtered tweets
- Respect rate limits (backoff, queuing)
- Build a small scheduler for timed posts

## The core idea

The hard part of a Twitter bot isn't the API — it's OAuth. Once you've done it once, every OAuth flow makes sense.

## Prerequisites

- CLI tool track recommended first
- TLS/HTTPS
