# Build Your Own Git

Content-addressable storage, trees, commits, refs, diff, merge.

## What you'll build

- Hash blobs with SHA-1 and store them in `.git/objects`
- Model trees and commits as blobs that reference other blobs
- Implement `init`, `add`, `commit`, `log`, `status`, `diff`
- Implement branches and `checkout`
- Implement a 3-way merge
- Talk to a remote via the smart HTTP protocol

## The core idea

Git feels magical until you realize it's a content-addressable filesystem with a nice CLI. Then it's obvious.

## Prerequisites

- Hashing, file I/O
- Data structures (trees)
