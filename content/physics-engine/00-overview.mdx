# Build Your Own Physics Engine

Rigid body dynamics, collision detection, constraint solving.

## What you'll build

- Integrate Newton's equations (semi-implicit Euler, then Verlet)
- Detect collisions: sphere-sphere, AABB, then SAT for convex polygons
- Resolve collisions with impulses
- Add friction and restitution
- Implement a sequential-impulse constraint solver for joints

## The core idea

A physics engine is: update positions, detect overlaps, apply corrective impulses. The art is doing it stably and fast.

## Prerequisites

- Linear algebra
- Floating-point gotchas
