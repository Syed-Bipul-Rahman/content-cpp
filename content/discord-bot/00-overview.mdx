# Build Your Own Discord Bot

Gateway WebSocket, REST, slash commands, voice basics.

## What you'll build

- Authenticate via Discord's OAuth
- Connect to the Gateway WebSocket and handle the heartbeat
- Dispatch events, respond to messages
- Register and handle slash commands
- Optional: voice connection and playback

## The core idea

Discord's protocol is a textbook modern bot protocol — WebSocket for events, REST for actions. Master it once and every similar API looks the same.

## Prerequisites

- IRC bot track recommended first
- JSON
- TLS/WebSocket
