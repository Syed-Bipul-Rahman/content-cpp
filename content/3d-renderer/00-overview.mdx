# Build Your Own 3D Renderer

Software rasterizer → OpenGL pipeline → basic ray tracer.

## What you'll build

- Build a 2D framebuffer and draw lines/triangles with Bresenham and edge functions
- Implement a complete software rasterizer with z-buffer and perspective
- Move to OpenGL and implement the same pipeline on the GPU
- Add Phong/Blinn-Phong lighting and texturing
- Build a toy ray tracer for comparison

## The core idea

Graphics is linear algebra made visible. We'll motivate every matrix with a picture.

## Prerequisites

- Math: vectors, matrices, basic trig
- Strong pointers/memory story
