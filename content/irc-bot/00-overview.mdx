# Build Your Own IRC Bot

Connect to IRC, listen, respond, with a plugin system.

## What you'll build

- Speak the IRC text protocol
- Connect, identify, join channels
- Parse messages and respond
- Dispatch to plugins (commands, triggers)
- Reconnect on drop

## The core idea

IRC is a line-based text protocol — the perfect training wheels for network bots before you touch WebSockets.

## Prerequisites

- Networking basics
