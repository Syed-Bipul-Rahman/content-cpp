# Build Your Own BitTorrent Client

Bencoding, trackers, peer handshake, piece selection.

## What you'll build

- Parse `.torrent` files (bencoding)
- Announce to trackers over HTTP
- Handshake and exchange pieces with peers
- Verify piece hashes
- Implement a reasonable piece-selection strategy

## The core idea

BitTorrent is a peer protocol dressed in a file format. Once you've written one, every P2P system feels approachable.

## Prerequisites

- Webserver track recommended first
- Hashing
