# Build Your Own Emulator / VM

CHIP-8 → NES → a tiny JIT. Fetch, decode, execute, and beyond.

## What you'll build

- Implement CHIP-8 — the 'hello world' of emulators. ~700 lines
- Move up to the 6502 and NES (picture processing unit, APU)
- Build a cycle-accurate vs instruction-level tradeoff
- Sketch a tiny JIT that compiles hot blocks to native code

## The core idea

An emulator is a program that runs another CPU in software. Once you can do CHIP-8, you can do anything — the techniques scale.

## Prerequisites

- Bitwise operations
- State machines
