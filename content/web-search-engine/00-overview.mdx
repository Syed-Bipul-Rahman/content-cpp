# Build Your Own Web Search Engine

A crawler, indexer, and ranked query engine for the open web.

## What you'll build

- Write a polite crawler that respects `robots.txt`
- Deduplicate and normalize URLs
- Extract text from HTML
- Build and persist a large-scale inverted index
- Implement PageRank on the link graph
- Serve ranked queries over HTTP

## The core idea

Crawling scales differently than indexing. Indexing scales differently than ranking. This track makes the seams visible.

## Prerequisites

- Search engine track
- Webserver track
