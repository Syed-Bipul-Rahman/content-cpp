# Build Your Own Augmented Reality

Camera input, fiducial marker tracking, pose estimation, overlay rendering.

## What you'll build

- Capture frames from a webcam
- Detect ArUco-style fiducial markers
- Recover 3D pose from 2D marker corners (PnP)
- Render a 3D model aligned to the marker in real time
- Extend toward natural-feature tracking (ORB)

## The core idea

AR is computer vision + 3D rendering, stitched by a pose. Each piece is approachable on its own.

## Prerequisites

- 3D renderer track recommended first
- Linear algebra
