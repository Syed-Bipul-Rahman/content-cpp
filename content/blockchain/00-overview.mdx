# Build Your Own Blockchain

A working proof-of-work coin with P2P gossip and a UTXO model.

## What you'll build

- Implement a block: header + transactions + hash
- Build a Merkle tree for transactions
- Implement proof-of-work mining
- Design UTXOs and transaction validation
- Gossip blocks and transactions over TCP
- Handle forks with longest-chain-wins

## The core idea

A blockchain is an append-only log where participants agree on the tip via cost. We'll build enough to mine a private coin.

## Prerequisites

- Hashing, signatures
- Networking basics
